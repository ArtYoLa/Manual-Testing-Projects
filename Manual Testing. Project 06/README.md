–°—É—â–µ—Å—Ç–≤—É–µ—Ç —á–µ—Ç—ã—Ä–µ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Å–ø–æ—Å–æ–±–∞ —Ä–∞–±–æ—Ç—ã API –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ç–æ–≥–æ, –∫–æ–≥–¥–∞ –∏ –ø–æ—á–µ–º—É –æ–Ω–∏ –±—ã–ª–∏ —Å–æ–∑–¥–∞–Ω—ã.

#### **SOAP API**

**SOAP** ‚Äî _Simple Object Access Protocol_, —Ç–æ –µ—Å—Ç—å –ø—Ä–æ—Å—Ç–æ–π –ø—Ä–æ—Ç–æ–∫–æ–ª –¥–æ—Å—Ç—É–ø–∞ –∫ –æ–±—ä–µ–∫—Ç–∞–º. SOAP –ø–æ–¥—Ä–∞–∑—É–º–µ–≤–∞–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ _WSDL (Web Services Description Language)_ ‚Äî —è–∑—ã–∫ –æ–ø–∏—Å–∞–Ω–∏—è –≤–µ–±-—Å–µ—Ä–≤–∏—Å–æ–≤ –∏ –¥–æ—Å—Ç—É–ø–∞ –∫ –Ω–∏–º, –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–π –Ω–∞ —è–∑—ã–∫–µ XML. –¢–æ –µ—Å—Ç—å –∫–ª–∏–µ–Ω—Ç –∏ —Å–µ—Ä–≤–µ—Ä –æ–±–º–µ–Ω–∏–≤–∞—é—Ç—Å—è —Å–æ–æ–±—â–µ–Ω–∏—è–º–∏ –ø–æ—Å—Ä–µ–¥—Å—Ç–≤–æ–º XML. –≠—Ç–æ –º–µ–Ω–µ–µ –≥–∏–±–∫–∏–π API, –∫–æ—Ç–æ—Ä—ã–π –±—ã–ª –ø–æ–ø—É–ª—è—Ä–µ–Ω –≤ –ø—Ä–æ—à–ª–æ–º.

#### **RPC API**

–¢–∞–∫–∏–µ API –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è —Å–∏—Å—Ç–µ–º–æ–π **—É–¥–∞–ª—ë–Ω–Ω–æ–≥–æ –≤—ã–∑–æ–≤–∞ –ø—Ä–æ—Ü–µ–¥—É—Ä**. –ö–ª–∏–µ–Ω—Ç –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é (–∏–ª–∏ –ø—Ä–æ—Ü–µ–¥—É—Ä—É) –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ, –∏ —Å–µ—Ä–≤–µ—Ä –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–±—Ä–∞—Ç–Ω–æ –∫–ª–∏–µ–Ω—Ç—É.

#### **Websocket API**

**Websocket API** ‚Äî —ç—Ç–æ –µ—â—ë –æ–¥–Ω–∞ —Å–æ–≤—Ä–µ–º–µ–Ω–Ω–∞—è —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∞ web API, –∫–æ—Ç–æ—Ä–∞—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –æ–±—ä–µ–∫—Ç—ã JSON –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö. WebSocket API –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω—é—é —Å–≤—è–∑—å –º–µ–∂–¥—É –∫–ª–∏–µ–Ω—Ç—Å–∫–∏–º–∏ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è–º–∏ –∏ —Å–µ—Ä–≤–µ—Ä–æ–º. –°–µ—Ä–≤–µ—Ä –º–æ–∂–µ—Ç –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å —Å–æ–æ–±—â–µ–Ω–∏—è –æ–±—Ä–∞—Ç–Ω–æ–≥–æ –≤—ã–∑–æ–≤–∞ –ø–æ–¥–∫–ª—é—á—ë–Ω–Ω—ã–º –∫–ª–∏–µ–Ω—Ç–∞–º, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –µ–≥–æ –±–æ–ª–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–º, —á–µ–º REST API.

#### **REST API**

–ù–∞ —Å–µ–≥–æ–¥–Ω—è—à–Ω–∏–π –¥–µ–Ω—å —ç—Ç–æ —Å–∞–º—ã–µ –ø–æ–ø—É–ª—è—Ä–Ω—ã–µ –∏ –≥–∏–±–∫–∏–µ API-–∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã –≤ –ò–Ω—Ç–µ—Ä–Ω–µ—Ç–µ. –ö–ª–∏–µ–Ω—Ç –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç –∑–∞–ø—Ä–æ—Å—ã –Ω–∞ —Å–µ—Ä–≤–µ—Ä –≤ –≤–∏–¥–µ –¥–∞–Ω–Ω—ã—Ö. –°–µ—Ä–≤–µ—Ä –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —ç—Ç–æ—Ç –∫–ª–∏–µ–Ω—Ç—Å–∫–∏–π –≤–≤–æ–¥ –¥–ª—è –∑–∞–ø—É—Å–∫–∞ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏—Ö —Ñ—É–Ω–∫—Ü–∏–π –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –≤—ã—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –∫–ª–∏–µ–Ω—Ç—É.

**REST (_Representational State Transfer_) API** ‚Äî –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π —Å—Ç–∏–ª—å –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è —Å–∞–π—Ç–æ–≤ –∏ –≤–µ–±-–ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π —Å —Å–µ—Ä–≤–µ—Ä–æ–º. REST –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –Ω–∞–±–æ—Ä —Ñ—É–Ω–∫—Ü–∏–π, —Ç–∞–∫–∏—Ö –∫–∞–∫ GET, POST, DELETE –∏ —Ç. –¥., –∫–æ—Ç–æ—Ä—ã–µ –∫–ª–∏–µ–Ω—Ç—ã –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –¥–æ—Å—Ç—É–ø–∞ –∫ –¥–∞–Ω–Ω—ã–º —Å–µ—Ä–≤–µ—Ä–∞. –ö–ª–∏–µ–Ω—Ç—ã –∏ —Å–µ—Ä–≤–µ—Ä—ã –æ–±–º–µ–Ω–∏–≤–∞—é—Ç—Å—è –¥–∞–Ω–Ω—ã–º–∏ –ø–æ –ø—Ä–æ—Ç–æ–∫–æ–ª—É HTTP.

–ì–ª–∞–≤–Ω–∞—è –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç—å REST API –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ —Ç–∞–∫–∞—è –ø–µ—Ä–µ–¥–∞—á–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –±–µ–∑ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è —Å–æ—Å—Ç–æ—è–Ω–∏—è. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å–µ—Ä–≤–µ—Ä—ã –Ω–µ —Å–æ—Ö—Ä–∞–Ω—è—é—Ç –∫–ª–∏–µ–Ω—Ç—Å–∫–∏–µ –¥–∞–Ω–Ω—ã–µ –º–µ–∂–¥—É –∑–∞–ø—Ä–æ—Å–∞–º–∏. –í–∞–∂–Ω–æ –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –∫–ª–∏–µ–Ω—Ç—Å–∫–∏–µ –∑–∞–ø—Ä–æ—Å—ã –∫ —Å–µ—Ä–≤–µ—Ä—É –≤ –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã URL-–∞–¥—Ä–µ—Å–∞–º, –∫–æ—Ç–æ—Ä—ã–µ –º—ã –≤–≤–æ–¥–∏–º –≤ –±—Ä–∞—É–∑–µ—Ä–µ –¥–ª—è –ø–æ—Å–µ—â–µ–Ω–∏—è –≤–µ–±-—Å–∞–π—Ç–∞, –∞ –æ—Ç–≤–µ—Ç –æ—Ç —Å–µ—Ä–≤–µ—Ä–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ø—Ä–æ—Å—Ç—ã–µ –¥–∞–Ω–Ω—ã–µ –±–µ–∑ —Ç–∏–ø–∏—á–Ω–æ–≥–æ –≥—Ä–∞—Ñ–∏—á–µ—Å–∫–æ–≥–æ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è –≤–µ–±-—Å—Ç—Ä–∞–Ω–∏—Ü—ã.

RESTful –≤–µ–±-—Å–µ—Ä–≤–∏—Å—ã, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –≥–æ—Ä–∞–∑–¥–æ –ø—Ä–æ—â–µ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å, —á–µ–º, –Ω–∞–ø—Ä–∏–º–µ—Ä, –≤–µ–±-—Å–µ—Ä–≤–∏—Å—ã –Ω–∞ –æ—Å–Ω–æ–≤–µ SOAP:

<h3 id="—Å–µ—Ç–µ–≤—ã–µ-–ø—Ä–æ—Ç–æ–∫–æ–ª—ã">–°–µ—Ç–µ–≤—ã–µ –ø—Ä–æ—Ç–æ–∫–æ–ª—ã</h3>

REST API, –∫–∞–∫ –±—ã–ª–æ —Å–∫–∞–∑–∞–Ω–æ –≤—ã—à–µ, —è–≤–ª—è—é—Ç—Å—è —Å–∞–º—ã–º–∏ –ø–æ–ø—É–ª—è—Ä–Ω—ã–º–∏ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞–º–∏ –≤ –ò–Ω—Ç–µ—Ä–Ω–µ—Ç–µ. –û–Ω–∏ —Ä–∞–±–æ—Ç–∞—é—Ç –Ω–∞ –±–∞–∑–µ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ HTTP (HTTPS). –û–¥–Ω–∞–∫–æ –µ—Å—Ç—å –∏ –¥—Ä—É–≥–∏–µ —Å–µ—Ç–µ–≤—ã–µ –ø—Ä–æ—Ç–æ–∫–æ–ª—ã.

**HTTP (HyperText Transfer Protocol)** ‚Äî –ø—Ä–æ—Ç–æ–∫–æ–ª –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ (–≥–∏–ø–µ—Ä—Ç–µ–∫—Å—Ç–∞), –Ω–∞ –±–∞–∑–µ –∫–æ—Ç–æ—Ä–æ–≥–æ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∏—Ä—É—é—Ç –≤—Å–µ —Å–µ–≥–æ–¥–Ω—è—à–Ω–∏–µ —Å–∞–π—Ç—ã. –í –µ–≥–æ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –≤—Ö–æ–¥–∏—Ç –ø—Ä–æ—Ü–µ—Å—Å –∑–∞–ø—Ä–∞—à–∏–≤–∞–Ω–∏—è –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã—Ö –¥–∞–Ω–Ω—ã—Ö —É –≤–∏—Ä—Ç—É–∞–ª—å–Ω–æ —É–¥–∞–ª–µ–Ω–Ω–æ–π —Å–∏—Å—Ç–µ–º—ã (—Ñ–∞–π–ª—ã, –≤–µ–±-—Å—Ç—Ä–∞–Ω–∏—Ü—ã –∏ –ø—Ä–æ—á–µ–µ).

**TCP (Transmission Control Protocol)** ‚Äî –æ–¥–∏–Ω –∏–∑ –æ—Å–Ω–æ–≤–Ω—ã—Ö –ø—Ä–æ—Ç–æ–∫–æ–ª–æ–≤ –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö –∏–Ω—Ç–µ—Ä–Ω–µ—Ç–∞, –∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è —Å–≤—è–∑—É—é—â–∏–º –∑–≤–µ–Ω–æ–º –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–∫–∏ –∫–∞—á–µ—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è –º–µ–∂–¥—É –¥–≤—É–º—è —É—Å—Ç—Ä–æ–π—Å—Ç–≤–∞–º–∏, –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö –∏ –≤–µ—Ä–∏—Ñ–∏–∫–∞—Ü–∏–∏ –∏—Ö –ø–æ–ª—É—á–µ–Ω–∏—è.

**IP (Internet Protocol)** ‚Äî –ø—Ä–æ—Ç–æ–∫–æ–ª, –≤ —Ñ—É–Ω–∫—Ü–∏–∏ –∫–æ—Ç–æ—Ä–æ–≥–æ –≤—Ö–æ–¥–∏—Ç –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç—å –¥–æ—Å—Ç–∞–≤–∫–∏ —Å–æ–æ–±—â–µ–Ω–∏–π –ø–æ –≤—ã–±—Ä–∞–Ω–Ω–æ–º—É –∞–¥—Ä–µ—Å—É. –ü—Ä–∏ —ç—Ç–æ–º –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –¥–µ–ª–∏—Ç—Å—è –Ω–∞ –ø–∞–∫–µ—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –ø–æ—Å—Ç–∞–≤–ª—è—Ç—å—Å—è –ø–æ-—Ä–∞–∑–Ω–æ–º—É.
